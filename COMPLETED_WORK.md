## 🎨 디자인 특징

### 리뷰어 메인 페이지 (리워딩스 스타일)
- **카드 기반 레이아웃**: 깔끔하고 직관적
- **이미지 중심**: 제품 사진이 눈에 띔
- **중요 정보 강조**:
  - 리뷰 포인트 (노란색 박스)
  - 신청 현황 (파란색 숫자)
  - 마감 임박 (빨간색 배지)
- **Hover 효과**: 카드에 마우스 올리면 이미지 확대 + 그림자
- **반응형**: 모바일/태블릿/데스크톱 모두 대응

### 캠페인 등록 페이지
- **단계별 카드**: 기본 정보 → 플랫폼 → 일정 → 선정 → 리뷰 가이드
- **이미지 미리보기 그리드**: 5개 이미지를 한눈에
- **삭제 버튼**: Hover 시 표시되는 X 버튼
- **대표 이미지 표시**: 첫 번째 이미지에 "대표" 배지
